d3-testbed

Try it

$ git clone git://github.com/doitian/d3-testbed.git
$ cd d3-testbed
$ make

Start a server:

$ python -m SimpleHTTPServer

or if you are using python 3.0+

$ python -m http.server

Then open the http://localhost:8000/test.html in browser. Use Left/Right or h/l to navigate.
